---
title: "Pseudo-Parameters"
description: "Pseudo-parameters available in the Azure Resource Manager lexicon"
---

The Azure Resource Manager lexicon provides **6** pseudo-parameters — predefined values available in every stack without explicit declaration.

| Name | Value |
|------|-------|
| `ResourceGroupName` | `Azure.ResourceGroupName` |
| `ResourceGroupLocation` | `Azure.ResourceGroupLocation` |
| `ResourceGroupId` | `Azure.ResourceGroupId` |
| `SubscriptionId` | `Azure.SubscriptionId` |
| `TenantId` | `Azure.TenantId` |
| `DeploymentName` | `Azure.DeploymentName` |
