The Flyway lexicon ships AI skills that teach AI coding agents how to build, validate, and deploy Flyway configurations from a chant project.

## What are skills?

Skills are structured markdown documents bundled with a lexicon. When an AI agent works in a chant project, it discovers and loads relevant skills automatically — giving it operational knowledge about the deployment workflow without requiring the user to explain each step.

## Installation

When you scaffold a new project with `chant init --lexicon flyway`, the skills are installed to `skills/` for automatic agent discovery.

To refresh skills after a lexicon update:

```bash
chant update
```

## Skill: chant-flyway

The `chant-flyway` skill covers the core Flyway configuration workflow:

- **Resource types** — FlywayProject, FlywayConfig, Environment, resolvers, provisioners
- **Build** — `chant build src/` to synthesize TOML configuration
- **Validate** — `chant lint src/` + post-synth checks
- **Composites** — when to use each composite pattern
- **Troubleshooting** — lint rule violations, configuration errors

## Skill: chant-flyway-migrations

The `chant-flyway-migrations` skill covers migration management patterns:

- **Migration lifecycle** — versioned, repeatable, and undo migrations
- **Naming conventions** — versioning schemes and descriptions
- **Baseline** — establishing baselines for existing databases
- **Callbacks** — pre/post migration hooks
- **Multi-schema** — managing migrations across schemas

## Skill: chant-flyway-security

The `chant-flyway-security` skill covers security best practices:

- **Credentials** — Vault integration, environment-based secrets
- **Access control** — least-privilege database users for migrations
- **Audit** — migration history and change tracking
- **Post-synth checks** — maps to relevant Flyway lint rules

## MCP integration

The lexicon also provides MCP (Model Context Protocol) tools and resources that AI agents can use programmatically:

| MCP tool | Description |
|----------|-------------|
| `build` | Build the chant project |
| `lint` | Run lint rules |
| `explain` | Summarize project resources |
| `scaffold` | Generate starter files |
| `search` | Search available resource types |

## LSP integration

The Flyway lexicon also provides LSP support via `chant serve lsp`:
- **Completions** — type `new ` to get Flyway resource completions
- **Hover** — hover over type names to see TOML serialization hints
