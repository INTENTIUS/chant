---
title: "Intrinsic Functions"
description: "Built-in intrinsic functions for the Flyway lexicon"
---

The Flyway lexicon provides **3** intrinsic functions.

| Function | Description | Output Key | Tag? |
|----------|-------------|------------|------|
| `resolve` | Resolver reference — $\{resolverName.key\} | `resolve` | No |
| `placeholder` | Built-in placeholder reference — $\{flyway:name\} | `placeholder` | No |
| `env` | Environment variable reference — $\{env.VAR_NAME\} | `env` | No |
