---
title: "Pseudo-Parameters"
description: "Pseudo-parameters available in the GCP Config Connector lexicon"
---

The GCP Config Connector lexicon provides **3** pseudo-parameters — predefined values available in every stack without explicit declaration.

| Name | Value |
|------|-------|
| `ProjectId` | `GCP::ProjectId` |
| `Region` | `GCP::Region` |
| `Zone` | `GCP::Zone` |
