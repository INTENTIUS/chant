## Synopsis

```
npx chant dev rollback [path] [--verbose]
```

## Description

`chant dev rollback` lists or restores previous generation snapshots for a lexicon. This allows reverting to an earlier generation if a new run introduces issues.

This is a **lexicon authoring** command — project users don't typically need it.

## Options

| Option | Description |
|--------|-------------|
| `-v, --verbose` | Show detailed snapshot information |
